## Prerequisites

Before you begin, ensure you have the following:

- A **WordPress** website with **WooCommerce** installed and activated.
- PHP version **7.2 or higher**.
- WordPress version **5.0 or higher**.
- **UPITranzact API credentials** (API Public Key, Secret Key).

## Step 2: Upload & Install the Plugin

1. Log in to your **WordPress Admin Dashboard**.
2. Navigate to **Plugins â†’ Add New â†’ Upload Plugin**.
3. Click **Choose File**, select the `.zip` file you downloaded, and click **Install Now**.
4. After installation, click **Activate Plugin**.

## Step 3: Configure UPITranzact Settings

1. Go to **WooCommerce â†’ Settings â†’ Payments**.
2. Click on **UPITranzact Payment Gateway** from the list of available payment methods.
3. Enable the payment method by checking **"Enable UPITranzact"**.
4. Enter your **Merchant Id**, **API Public Key** and **Secret Key** provided by UPITranzact.
5. Customize any additional settings like title, description, success/failure redirection if required.
6. Click **Save Changes**.

## Step 4: Go Live

1. Make sure your **live API credentials** are saved correctly.
2. You're ready to accept UPI payments on your WooCommerce store!
